# bakfilehandler
A .bak file handler/manager for Windows

## Usage
Run the script, allow admin, press [I] to install. ([R] to remove/uninstall)

Then you can "run" a .bak file, with options to :

Update (original file --> backup);

Restore (file <-- backup);

Restore & delete;

Open folder;

or Cancel.

That's all.
